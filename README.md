# Sistema de Cadastro de Clientes

Sistema desenvolvido em Python utilizando SQLite, com foco em
manutenção de dados, regras de negócio e organização de código.

## Tecnologias
- Python
- SQLite
- Git

## Funcionalidades
- Cadastro, listagem, atualização e exclusão lógica
- Validações de dados
- Estrutura em camadas

## Objetivo
Simular um sistema interno corporativo semelhante a ambientes
Delphi ou PowerBuilder.
